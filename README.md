# tubelleza

Comparador de productos de belleza inspirado en el estilo de Las Margaritas.

## Cómo correrlo

```bash
npm install
npm run dev
```

Abre http://localhost:5173

## Cómo funciona

1. Buscá productos en la barra superior (nombre, marca o categoría).
2. Ajustá los pesos de **calificación**, **reseñas** y **precio bajo** en el panel lateral.
3. El ranking se reordena en vivo y muestra al ganador con medalla.

Los datos están en `src/data/products.js` — son de ejemplo, listos para reemplazar por una API real.
